# Business UP - Multi-Category Affiliate Hub

Sistema di lead generation e affiliate management multi-categoria con Telegram bot + Mini App.

## Stack
- **Backend**: Supabase (Postgres + Edge Functions Deno)
- **Bot**: Telegram Bot API  
- **Frontend**: GitHub Pages (Mini App)
- **CI/CD**: GitHub Actions → Auto-deploy

## Setup
1. Crea progetto Supabase
2. Aggiungi secrets GitHub
3. Push → Deploy automatico
